This is the starting line. By the end of this course you will know what Ryu is, have it running on
your machine, and have held a real conversation with an agent that uses tools, all without an API
key or a terminal.

## Who this is for

Everyone. You do not need to be a developer, and you do not need to bring your own models or keys.
Ryu ships a complete local stack that works the moment you install it.

## What you will learn

- What Ryu actually is, and why "the car around the engine" matters
- How to install Ryu and what happens on first run
- How agents work, and how to meet the default **Ryu** agent
- How to run your first chat, including tool use
