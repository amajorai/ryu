The Essentials course got you installed and chatting with the default Ryu agent. This track takes you under the hood. You will learn how Ryu is assembled from swappable parts, and how to reshape every one of them to fit your work.

The theme of this course is one idea Ryu takes seriously: nothing is hardcoded. Every model, engine, skill, tool, and policy is a default you can change.

## Who this is for

- People who finished the Essentials course (install, the default Ryu agent, a first chat).
- Power users setting Ryu up for themselves.
- Anyone configuring Ryu for a small team.

You do not need to write code. Everything here happens in the desktop app, reached from the sidebar or the command palette (Ctrl+K on Windows, Cmd+K on macOS).

## What you will learn

- How an agent is built from independent slots, like a trading card.
- How engines and models are chosen, installed, and swapped.
- How skills and tools extend what an agent can do, and how the Gateway governs them.
- How Spaces and memory give an agent private, recallable knowledge.
- How to automate recurring work with schedules, monitors, delegation, and workflows.
