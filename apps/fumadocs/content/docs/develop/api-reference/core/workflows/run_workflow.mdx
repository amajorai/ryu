          Routes through the durable engine selected by
          `durable::select_engine()`:
            - `FallbackEngine` (default): in-process petgraph topological executor with
              file-backed resumable state; crash-recoverable at the node checkpoint level.
            - `RestateEngine` (opt-in via `RYU_RESTATE_ENABLED=1`): each node step runs
              inside Restate's durable journal, surviving full process kills.

          Returns the same `WorkflowRun` shape regardless of backend (AC1).

          Returns 503 when the gateway is unreachable and fail-closed is in
          effect (AC3).
