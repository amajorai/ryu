      - content: >-
          This route is **public** (it sits outside `require_auth`) because an
          external

          Composio delivery cannot send Core's bearer token. It is instead
          authenticated

          **fail-closed** with an HMAC-SHA256 signature over the raw body keyed
          by

          `COMPOSIO_WEBHOOK_SECRET` (see
          [`crate::composio_triggers::verify_webhook_signature`]):

          when the secret is unset, or the `webhook-signature` header is
          absent/invalid,

          the request is rejected with 401 and nothing fires.
