          Always responds `200`. When the gateway is unreachable it returns

          `{ "reachable": false, ... }` rather than an error status, so the
          desktop

          status spine can distinguish "gateway down (Core up)" from "Core down"
          — the

          typed client treats any non-2xx as Core being unreachable.


          Forwards the gateway bearer token (`RYU_GATEWAY_TOKEN`) when
          configured, in

          case the gateway runs with auth enabled.
