          `capabilities` gate ({ inline, offline }) selecting which surface may
          offer it and an `enforced` honesty flag (true only when wired to real
          execution). The built-in seed table is merged with any user-authored
          `custom_evaluators` from config (a custom entry overrides a built-in
          by `id` and reports `builtin: false`). Read-only; not behind the admin
          gate.
