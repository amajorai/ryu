This page links to the authoritative design documents that drive Ryu's architecture and
development. These live in the repository root `docs/` directory and are the source of truth for
design decisions, specs, and audit findings.

<Callout type="info">
These documents are internal design artifacts, not user-facing guides. They are linked here so
agents and developers can find them without navigating the repository root. For user-facing
documentation, see the relevant sections in the sidebar.
</Callout>
