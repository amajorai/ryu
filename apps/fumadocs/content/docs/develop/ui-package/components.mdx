## Editor

The PlateJS editor is vendored at `@ryu/ui/components/editor` with 45+ plugin kits and 80+ UI components. See the editor directory for the full inventory:

```tsx
import { PlateEditor } from "@ryu/ui/components/editor/plate-editor";
import { editorKit } from "@ryu/ui/components/editor/editor-kit";
```
