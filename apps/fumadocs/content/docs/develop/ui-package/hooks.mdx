All hooks export from `@ryu/ui/hooks/<name>`.

## Data & State

| Hook | Import | Description |
|---|---|---|
| **useDataGrid** | `@ryu/ui/hooks/use-data-grid` | The main data grid hook (2000+ lines). Manages TanStack Table state, cell editing, selection, column ordering, sorting, filtering, and clipboard paste. |
| **useControllableState** | `@ryu/ui/hooks/use-controllable-state` | Controlled/uncontrolled state pattern. Accepts an optional initial value and an onChange callback. |
| **useMounted** | `@ryu/ui/hooks/use-mounted` | Returns `true` after the component mounts. Useful for SSR-safe client-only rendering. |
| **useIsomorphicLayoutEffect** | `@ryu/ui/hooks/use-isomorphic-layout-effect` | `useLayoutEffect` on the client, `useEffect` on the server. |

## Input & Interaction

| Hook | Import | Description |
|---|---|---|
| **useDebouncedCallback** | `@ryu/ui/hooks/use-debounced-callback` | Debounced callback that cancels on unmount. |
| **useDebounce** | `@ryu/ui/hooks/use-debounce` | Debounced value (returns the value after a delay). |
| **useLazyRef** | `@ryu/ui/hooks/use-lazy-ref` | Ref that initializes its value lazily (only on first render). |
| **useCallbackRef** | `@ryu/ui/hooks/use-callback-ref` | Stabilizes a callback ref across renders. |
| **useAsRef** | `@ryu/ui/hooks/use-as-ref` | Converts props to a ref. |
| **useProximityHover** | `@ryu/ui/hooks/use-proximity-hover` | Proximity-based hover detection — triggers when the cursor is near, not just over the element. |
| **useHoverCapable** | `@ryu/ui/hooks/use-hover-capable` | Detects whether the device truly supports hover (not just touch). |
| **useIsTouchDevice** | `@ryu/ui/hooks/use-is-touch-device` | Returns `true` on touch devices. |
| **useMobile** | `@ryu/ui/hooks/use-mobile` | Breakpoint detection — returns `true` below the mobile threshold. |
| **useBadgeOverflow** | `@ryu/ui/hooks/use-badge-overflow` | Detects when badge content overflows its container. |

## Usage pattern

```tsx
import { useDebouncedCallback } from "@ryu/ui/hooks/use-debounced-callback";
import { useMobile } from "@ryu/ui/hooks/use-mobile";
import { useControllableState } from "@ryu/ui/hooks/use-controllable-state";

function SearchInput({ value, onChange }: Props) {
  const isMobile = useMobile();
  const debouncedChange = useDebouncedCallback(onChange, 300);

  return (
    <input
      onChange={(e) => debouncedChange(e.target.value)}
      placeholder={isMobile ? "Search…" : "Search documentation…"}
    />
  );
}
```
