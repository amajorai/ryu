Everything you do once Ryu is installed. This realm covers the surfaces you touch every day: the
desktop app and chat, the agents and teams that do the work, the engines and models that run them,
the catalogs you install from, and the productivity tools built on top.

New here? Start with the [User Guide](/docs/using-ryu/user-guide) for a tour of the app, then pick
what your agents run on in [Engines](/docs/using-ryu/engines) and [Models](/docs/using-ryu/models).
