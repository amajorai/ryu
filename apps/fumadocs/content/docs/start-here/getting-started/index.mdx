**Agents are powerful. Using them shouldn't be.** Ryu is the whole car built around the
agent engines that already exist (OpenAI, Claude Code, Gemma, any OpenAI-compatible runtime).
It bundles a local agent runtime, a chat UI, and an orchestration layer so you can pick an
agent and go, with no terminal, no API keys, and no MCP wiring required.

The desktop app talks to **Ryu Core**, a local backend that manages agent runtimes
(sidecars), routes your chat to the right engine, and keeps your conversations and memory on
your machine. Everything runs locally and privately by default.

## System requirements

- **Operating system** - macOS or Windows 10 and newer.
- **Memory** - 4 GB RAM minimum for the bundled local model. More RAM (and a GPU) lets you
  run larger models from the catalog.
- **Disk** - a few GB free for the local model and any agent runtimes you install.

<Callout type="info">
  Ryu works fully offline with the built-in agent. You only need network access to install
  additional models, skills, or cloud-backed agent runtimes.
</Callout>

## Install the desktop app

The fastest path is the prebuilt desktop app:

1. Download the Ryu desktop installer for your platform from [ryu.com/download](https://ryu.com/download).
2. Run the installer and launch Ryu.
3. Sign in, or continue locally.

### Run from source

If you have the monorepo checked out, you can run the desktop app directly with Bun:

```bash
bun install
bun run dev:desktop
```

This starts the Tauri desktop app, which spawns Ryu Core on `:7980` for you.

## What you get on first launch

- **A built-in agent (Pi).** Pi is installed by default and runs locally through llama.cpp
  with a bundled Gemma model, so you can chat immediately without any API key.
- **The local sidecar manager.** Ryu Core can download, verify, install, and start additional
  agent runtimes (Claude Code, Codex, Gemini CLI) on demand from the Services page.
- **Local conversations and memory.** Your chat history and long-term memory are stored on
  your machine.
