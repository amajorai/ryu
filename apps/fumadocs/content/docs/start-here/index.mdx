This section takes you from zero to a working agent. You will install the desktop app, learn the
two-service split that every request flows through, and understand how Ryu stays engine-agnostic
by design. Nothing here is aspirational - every step maps to a surface you can open today.

<Callout type="info">
  New to Ryu? Start with [Getting Started](/docs/start-here/getting-started) and come back to
  Architecture once you have sent your first message.
</Callout>
