**Agents are powerful. Using them shouldn't be.**

Ryu is end-to-end managed infrastructure for AI agents, the whole car built around the engines that
already exist (OpenAI, Claude Code, Gemma, any OpenAI-compatible runtime).

## What Ryu offers

- **For individuals and small teams** - Download, pick an agent, go. No terminal, no API keys, no MCP
  wiring.
- **For developers** - One config change puts a gateway firewall in front of any agent you already
  run.
- **For businesses** - We audit, build, deploy, and host; ship bots into Telegram, Slack, WhatsApp,
  and Discord.
